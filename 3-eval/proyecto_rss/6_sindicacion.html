Sindicación

La sindicación de contenidos es una forma de distribuir información de manera automática. Utiliza archivos en formato RSS o Atom para que los usuarios puedan recibir actualizaciones de un sitio web sin tener que visitarlo constantemente.

Es muy útil para seguir noticias, blogs o novedades de cualquier página. Solo necesitas un lector de feeds (como Feedly, Inoreader o incluso algunos navegadores web).
